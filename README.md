NES emulator made by a cat using SDL2 and ImGui, inspired by Mesen2.

Supported mappers:
- NROM (Mapper 0)
- MMC1 (Mapper 1)
- UxROM (Mapper 2)
- CNROM (Mapper 3)
- MMC3 (Mapper 4) (WIP)
- MMC2 (Mapper 9) (WIP)
- SunSoft FME-7 (Mapper 69) (WIP)
- J.Y. Company ASIC (Mapper 90, 209 and 211) (WIP)

Unimplemented mappers will default to MMC1 behavior.
