MeowNES is a completely broken NES Emulator made by a Cat, scrolling may be broken in some games.

Uses nes_ntsc for the NTSC Video Filter which I got from Mesen2, couldn't find the original source. It is modified to be used with MeowNES though.

Supported mappers:
- NROM (Mapper 0)
- MMC1 (Mapper 1)
- UxROM (Mapper 2)
- CNROM (Mapper 3)
- MMC3 (Mapper 4)
- MMC2 (Mapper 9)
- Nina01 (Mapper 34, SubMapper 1) (WIP)
- BNROM (Mapper 34, SubMapper 2)
- Namco163 / Namco129 (Mapper 19) (WIP)
- SunSoft FME-7 (Mapper 69)
- J.Y. Company ASIC (Mapper 90, 209 and 211) (WIP)
